> v0 / Lovable / Claude / Figma Make 등 어떤 UI 생성 툴에도 붙여넣어 쓸 수 있는 브리프.
> 한글 UI, 모바일 우선(PWA). 기획 상세는 `docs/기획서.md` 참고.
> v1(초안)에서 추가된 것: **문제 단위 커뮤니티 · 영상 로그 · 난이도 정규화 · "뭐부터 풀지" 추천.**

## 붙여넣기용 프롬프트 (여기부터 복사)

You are a senior product designer. Design a **mobile-first web app (PWA)** called **NewSetter**.
All UI text is in **Korean**. Output production-ready React + Tailwind components, one screen per artboard, sized for a **390px-wide phone**. Light mode first, but pick colors that also survive dark mode.

### Product (한 문장)
클라이밍 크루가 (1) 벽 세팅(뉴셋, 보통 4주 주기) 기준으로 "우리 크루가 이 암장을 이번 셋에 갔는지" 추적하고, (2) 언제·어디 갈지 투표로 정하고, (3) **각 문제(색=난이도)마다 완등 영상·베타·체감 난이도를 모아**, (4) 그 데이터로 **"다른 암장에서도 내 수준이 어디인지 / 같은 난이도 중 뭐부터 풀지"를 추천**하는 커뮤니티.

### 핵심 컨셉 3가지 (디자인이 살려야 할 것)
1. **뉴셋 트래킹** — 암장 카드에 "뉴셋 D+3 · 우리 크루 미방문" 상태 배지가 항상 붙는다.
2. **저마찰 기여 루프** — 방문 후 "뭐 풀었어?"를 원탭(완등/체감/꿀)으로 기록. 영상은 선택 첨부. 이게 앱의 엔진.
3. **개인화 난이도** — "여기 빨강 = 너의 홈 파랑 수준", "빨강 중 이게 제일 쉬움", "너 지금 이거 풀 만해" 를 자연스럽게 노출.

### Visual direction
- **Vibe**: 깔끔하고 활동적인 아웃도어 스포츠 커뮤니티. 정보 밀도는 높되 잘 읽히게. 플랫(그라디언트·큰 그림자 지양).
- **Base**: 초크 화이트 배경 #FAFAF7, 그래파이트 텍스트 #1C1C1A, 웜그레이 보더 #E7E5DE.
- **Accent (primary)**: 홀드 그린 **#2FB56A** (버튼·활성 탭·선택 상태).
- **의미 색상**: 방문/완등=그린 #2FB56A · 뉴셋 미방문(주목)=코랄 #F0663C, 앰버 #E0921A · 정보=블루.
- **홀드 색 표현**: 문제 색(흰/노/주/초/파/빨/검 등)은 실제 색 점(dot)/칩으로. 텍스트 색상과 헷갈리지 않게 원형 스와치 사용.
- **모서리**: 카드 16px, 컨트롤 10px. 폰트 Pretendard/시스템 산세리프. **문장형(sentence case), 이모지 최소.** 얇은 라인 아이콘(Lucide/Tabler).

### 정보 구조 (하단 탭 5개)
`홈 · 탐색 · ➕기록 · 캘린더 · 프로필`
- **가운데 ➕기록**은 강조된 원형 버튼 = **완등/영상 기록**(기여 루프의 핵심 진입점).
- 활성 탭만 accent 색.

### 컴포넌트 인벤토리 (재사용)
헤더바 · 하단 탭바 · 카드 · **상태 배지(pill)** · **홀드 색 스와치(dot)** · **체감 난이도 미터(가로 바: 쉬움↔어려움)** · **완등률 바** · **꿀 배지** · 리스트 아이템 · **영상 썸네일 카드(재생 오버레이)** · 아바타 · 세그먼트 토글(쉬움/적정/어려움) · 실력 칩(θ 레벨).

### 화면 목록 (14)

**A. 온보딩 · 크루**
1. **로그인** — 카카오 로그인 버튼, 로고, 한 줄 소개.
2. **크루 시작** — "새 크루 만들기"(primary) + "초대 코드로 참여"(입력+버튼).
3. **크루 만들기** — 이름 / 한 줄 소개 / 주 활동 지역 / 오픈카톡(선택).
4. **멤버 초대** — 초대 링크 카드 + [링크 복사]/[카톡 공유], 멤버 목록(역할 배지), "가입 신청 N명" 승인/거절(✓/✗).

**B. 홈 · 투표 · 방문**
5. **크루 홈** — ①다가오는 확정 일정 ②진행 중 투표(참여하기) ③"가야 할 암장"(뉴셋 미방문 순) ④**"오늘 추천"**(내 수준 기준 할 만한 문제 미리보기).
6. **투표 참여** — 위 "1. 가능한 날짜"(요일+시간대 라벨 체크, 득표수) → 아래 "2. 선호 암장"(체크, 각 암장에 "뉴셋 D+n · ★평점 · 방문여부").
7. **크루 캘린더** — 월 달력에 방문(그린)·예정(블루), 아래 이번 달 방문 리스트.

**C. 암장 · 문제 · 난이도**
8. **암장 상세** — 별점/리뷰수, 최근 세팅일(D+n) + "우리 크루 이번 셋 방문 ✓", 인스타 뉴셋 공지 링크, **난이도 보정 카드**("여기 빨강 ≈ 너의 홈 파랑"), 하단 큰 버튼 [이번 셋 문제 보기].
9. **문제 목록 (이번 셋)** — **색(난이도)별 섹션**. 각 섹션 안에서 **체감 쉬운 순으로 정렬**. 문제 행마다: 홀드 색 dot + 라벨, **체감 난이도 미터**, 완등률, 꿀 배지, 영상 n개 표시. 상단에 "쉬운 순 / 어려운 순" 정렬 토글.
10. **문제 상세** — 홀드 색·라벨·체감난이도 합의, 완등률, **완등 영상 피드(썸네일 그리드)**, 텍스트 베타 리스트, 하단 [내 기록 남기기].

**D. 기여 · 추천 · 나**
11. **완등 기록(➕)** — **저마찰 핵심 화면.** 암장/세트 선택 → 문제 고르기 → 각 문제에 원탭: 완등여부 토글 · 체감(쉬움/적정/어려움 세그먼트) · 꿀 토글 · (선택) 영상 첨부/촬영. **여러 문제 연속 추가**("영상 7개" 시나리오)에 최적화 — 방금 방문에서 푼 걸 죽 나열해 빠르게 체크.
12. **오늘 추천** — "너 지금 이거 풀 만해." 내 실력(θ) 기준 **미완등 & 할 만한 문제를 쉬운 순**. "성장 모드"(살짝 위 난이도) 토글. 각 카드에 예상 난이도·꿀 여부.
13. **난이도 지도(내 기준)** — 여러 암장 색을 공통 척도 위에 배치. "홈짐 빨강 = A짐 파랑 = B짐 노랑" 을 한눈에.
14. **프로필** — 아바타/닉네임, **내 실력 칩(θ 레벨)** + 홈짐/기준 색, 내 완등 로그(영상 포함) 타임라인, 설정.

### 상태·빈 화면
- 데이터 없을 때(콜드스타트): 문제 목록/추천은 "아직 기록이 적어요 — 색 기준으로 보여드려요" 안내 + 색=난이도 거친 정렬.
- 영상 업로드 중 진행 표시. 영상 없는 로그도 정상 노출(썸네일 자리에 색 dot).

### Microcopy tone
존댓말, 동사 우선 버튼("완등 기록", "영상 올리기", "이 문제 풀기"). 느낌표 남발 금지. 추천 문구는 단정 말고 제안형("이 정도면 풀 만해요").

### Deliverable
각 화면을 개별 컴포넌트로. 공통 컴포넌트(탭바·카드·배지·색 스와치·난이도 미터·영상 카드·세그먼트 토글)는 분리해 재사용. Tailwind 색은 위 hex를 토큰화(hold/coral/amber/chalk/graphite).
