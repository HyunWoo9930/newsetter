# 게시물 02 — 쿨타임 카루셀 (완성본)

> 이미지: `public/brand/carousel/slide-01.png` ~ `slide-07.png` (1080×1350, 4:5)
> 릴스 #1 "클라이밍하다 불편해진 개발자"와 같은 테마·톤. 릴스와 짝으로 운영.
> 재생성: HTML `scripts/reels/`(carousel 폴더는 세션 스크래치) → Playwright 스크린샷.

## 슬라이드 구성
1. 표지 — "다들 이런 경험, 있으시죠?" + 인사 초크볼 + 밀어서 보기
2. 문제1 — "암장 벽은 매주 하나씩 바뀌어요" + 4주 벽 달력
3. 문제2 — "근데… 나 여기 언제 왔더라?" + 폰뒤짐 초크볼
4. 저장 슬라이드 — "'홈 암장 쿨타임' 놓치면 생기는 일" 3가지
5. 해결1 — "홈 암장 간 지 몇 주째인지 열자마자" + 홈 화면 + 유레카
6. 해결2 — "크루 일정도 투표로" + 투표 화면 + 포인팅
7. CTA — "우리 동네 클라이밍, 한곳에서" + 완등 초크볼 + 프로필 링크

## 캡션 (그대로 복붙 가능 — 톤 확인하고 고쳐 쓰세요)

```
클라이밍하다 보면 다들 이런 경험 있으시죠? 🧗

암장 벽은 보통 매주 하나씩 바뀌고,
한 달이면 거의 모든 문제가 새로 세팅돼요.

근데 정작 "내가 여기 마지막에 언제 왔더라?"는
아무도 안 알려주더라고요.
결국 매번 내 인스타를 뒤지게 되죠. (벌써 n번째…)

그래서 그냥 만들어버렸습니다.
· 홈 암장 간 지 몇 주째인지 열자마자 확인
· 크루 일정은 안 되는 날만 체크 → 겹치는 날 자동 확정
· 암장 리뷰까지 한곳에서

웹에서 무료로 써보실 수 있어요 → 프로필 링크
저장해두고, 같이 다니는 크루원 태그해주세요 🙌
여러분은 "언제 갔더라"를 어떻게 해결하세요? 댓글로 알려주세요!
```
