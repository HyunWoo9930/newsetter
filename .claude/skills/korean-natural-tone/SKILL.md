# 한국어 네이티브 어투 (뉴세터 프로젝트)

## 왜 이 스킬이 필요한가

AI가 쓴 한국어 홍보 문구는 문법이 완벽해도 "번역체 + 마케팅체"라서 받는 사람이 3초 만에 광고로 분류하고 닫는다. 특히 인스타 DM은 개인 간 대화 공간이라, 조금이라도 '문안(文案)' 냄새가 나면 답장률이 급락한다. 이 스킬의 목표는 문구를 "잘 쓴 글"이 아니라 **"실제 사람이 손으로 친 메시지"**로 만드는 것.

## 이 프로젝트 사용자의 실제 어투 (기준점)

사용자가 직접 쓴 릴스 대본에서 추출한 톤 — 새 문구를 쓸 때 이 결을 따른다:

- "다들 이런 경험 있으시지 않으셨나요..?" — 물음표 앞 말줄임, 부드러운 동의 유도
- "너무 어렵지 않으신가요?!" — ?! 조합으로 공감 호소
- "만들어봤습니다!" — 자랑이 아니라 '시도해봤다'는 겸손형
- "사라져버려요 ㅜㅜ" — ㅜㅜ 한 번 정도, 과하지 않게
- "이것만 불편한게 아닌데요,," — 쉼표 두 개(,,)로 여운

핵심: **존댓말 + 혼잣말 섞인 구어체 + 가벼운 자기낮춤**. 반말·과장·영업 멘트 없음.

## 번역체/마케팅체 → 네이티브 변환 규칙

| 지우기 (번역체·마케팅체) | 바꾸기 (실제 어투) |
|---|---|
| ~하실 수 있습니다 | ~할 수 있어요 / ~돼요 |
| ~을 제공합니다 / 지원합니다 | ~돼요 / ~있어요 |
| 지금 바로 / 놓치지 마세요 | (삭제 — 재촉은 광고 신호) |
| 다양한 / 간편하게 / 손쉽게 | (삭제하거나 구체적 사실로 대체) |
| 안녕하세요! 저는 ~입니다. 다름이 아니라 | 안녕하세요! (자기소개는 한 줄, 필요할 때만) |
| · 기능1 · 기능2 · 기능3 (불릿 나열) | DM에서는 문장으로 풀어쓰기. 불릿은 게시물에서만 |
| 무료로 이용 가능합니다 | 무료예요 / 돈 안 들어요 ㅎㅎ |
| 피드백 부탁드립니다 | 별로면 별로라고 해주셔도 돼요 ㅎㅎ |

## 실제 한국인 인스타 DM의 특징

1. **문장이 짧고, 완결이 덜 됨.** "그래서요..." "그게 좀 그렇더라고요" 같은 흐림이 자연스러움.
2. **쿠션어 필수**: "혹시", "괜찮으시면", "갑자기 DM 드려서 죄송한데", "부담 없이".
3. **ㅎㅎ / ㅠㅠ / ㅋㅋ 는 조미료** — 한 메시지에 2~3번이면 충분. 이모지(🧗🙏)는 0~2개.
4. **자기낮춤이 신뢰를 만든다**: "허접하지만", "아직 부족한데", "테스트 삼아".
5. **링크는 맨 끝에 딱 하나.** 링크가 먼저 나오면 스팸.
6. **상대방 이야기로 시작**: 내 소개보다 상대 계정/크루 언급이 먼저. "○○크루 영상 잘 보고 있어요"가 첫 줄.
7. **부탁이 아니라 제안으로 끝**: "써주세요" 대신 "구경만 해보셔도 돼요".

## 작업 순서

1. 문구의 용도(DM/캡션/오픈채팅/댓글)와 받는 사람을 확인한다.
2. 초안을 쓴 뒤, 위 변환표로 번역체·마케팅체를 지운다.
3. 소리 내어 읽는다고 상상: "친구한테 이렇게 보내면 어색한가?"가 최종 기준.
4. 사용자에게 보여줄 때는 완성문 + "여기서 톤 더 낮추거나 높일 수 있어요" 옵션 제시.
5. 사용자가 고친 문구가 있으면 그 문구가 항상 우선 — 다음 문구부터 그 결을 학습해 반영한다.

## 예시

**Before (마케팅체):**
> 안녕하세요! 클라이밍 크루를 위한 웹 서비스 뉴세터입니다. 홈 암장 쿨타임 확인, 크루 일정 투표, 암장 리뷰 기능을 제공합니다. 지금 바로 무료로 이용해보세요!

**After (네이티브):**
> 안녕하세요! 저도 클라이밍하는 개발자인데요 ㅎㅎ
> 크루 일정 잡을 때마다 단톡에서 "언제 돼?" 무한반복하는 게 너무 불편해서... 그냥 제가 하나 만들어봤어요.
> 홈암장 간 지 몇 주째인지도 바로 보이고요.
> 부담 없이 구경만 해보셔도 좋아요! 별로면 별로라고 해주셔도 되고요 ㅎㅎ
