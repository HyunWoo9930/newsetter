# 카루셀 시리즈 3·4·5탄 기획 — "클라이밍하다 불편해진 개발자"

> 1탄 = 쿨타임([게시물02](게시물02_쿨타임_카루셀_완성.md)) · 2탄 = 일정지옥([게시물03](게시물03_일정지옥_카루셀_완성.md))에 이어지는 3편.
> 파일 번호로는 게시물04·05·06. 이미지 폴더는 `public/brand/carousel3` `carousel4` `carousel5`.
> 템플릿·톤·제작법은 기존과 동일: `scripts/reels/carousel2-slides.html` 변형 → Playwright 스크린샷.

## 왜 이 3개인가 (시리즈 설계)

| 탄 | 주제 | 타겟 | 앱 근거 기능 (실존 확인됨) | 핵심 지표 |
|---|---|---|---|---|
| **3탄** | "오늘 누구 가?" — 일정 잡아도 막상 누가 가는지 모름 | 크루원 (2탄 모멘텀 유지) | 캘린더 내일정 구분 · 세션 상세 시트(누가 가는지) · 참석/변경/취소 · SSE 실시간 알림 | 태그 수 + 프로필 방문 |
| **4탄** | "내 완등 영상 다 어디 갔지" — 기록이 흩어짐 | 개인 클라이머 전체 (1탄 콜백) | 완등 로그 + 영상 첨부 · 이번 셋 문제(색=난이도) · 체감 난이도 · 프로필 통계 · 성장 모드 | 저장 수 |
| **5탄** | "저 크루 없는데요,,?" — 개인 모드 | **크루 없는 클라이머** (1~4탄에서 소외됐던 층으로 확장) | 개인 모드 · 즐겨찾기 ★ = 홈 암장 · '가야 할 암장'(간 지 오래된 즐겨찾기) · 지도 탐색 | 팔로우 전환 + 가입 |

**설계 논리:**
- 2탄(크루 일정)이 크루장·크루원을 데려왔으니 → 3탄은 그 관심을 이어받아 **일정 확정 이후의 페인**("잡았는데 누가 와?")을 침. 2탄 표지("단톡에서 제일 무서운 말") 콜백으로 시리즈 정체성 강화.
- 4탄은 다시 개인 페인으로 — 1탄의 "결국 내 인스타를 뒤짐"을 그대로 콜백해서 시리즈 수미상관. 저장 지표가 제일 잘 나올 소재(기록 팁성 콘텐츠).
- 5탄은 오디언스 확장 — 지금까지 "크루" 얘기 보며 '난 크루 없는데'라고 생각한 사람에게 직접 말 걸기. 댓글에서 실제로 이 반응이 나오면 5탄을 앞당겨도 됨.

**난이도 환산 편(그로스 플랜 필러 A)을 뺀 이유:** 앱의 '난이도 지도'가 아직 "준비 중" 상태라(ClimbCrewApp.tsx 확인) 카루셀로 약속하면 오버프로미스. 기능 나오면 **6탄 "파랑이 파랑이 아님"**으로 — 리드마그넷(환산표)과 묶어서 시리즈 최강 카드로 아껴둠.

## 게시 순서·간격
```
3탄 게시 → 2~3일 → (릴스 #4 "누구 오늘 가?") → 3~4일 → 4탄 → 2~3일 → 5탄
```
