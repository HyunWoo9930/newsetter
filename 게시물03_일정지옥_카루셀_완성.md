> 시리즈 2탄. 타겟: 크루장·크루원. DM 시딩 메시지와 같은 페인("언제 돼? 핑퐁")으로 연결.
> [7]에는 이번에 실제로 추가한 🌱 기능(한 달간 아무도 안 간 암장 추천, PR #1) 실화면이 들어감.

## 슬라이드 구성
1. 표지 — "크루 단톡에서 제일 무서운 말" + 말풍선 "우리 언제 가?" + 폰뒤짐 초크볼
2. 단톡 핑퐁 재현 — 민석/서연/준호 말풍선 6개 + "어디서 많이 본 대화 아닌가요..?"
3. 결말 — "그렇게 이번 주도 아무도 못 갔어요,," + 지친 초크볼
4. 새 페인 — "암장 고르기는 더 어려워요..?" (각자 따로 다녀서) + 포인팅 초크볼
5. 저장 — "크루 일정, 왜 맨날 무산될까요..?" 3가지 + 📌 저장
6. 해결1 — "안 되는 날만 X 치면 끝!" + 투표 캘린더 화면 + 유레카
7. 해결2 — "한 달간 아무도 안 간 암장엔 🌱 새싹이 붙어요" + 실제 앱 화면 + 인사 초크볼
8. CTA — "일정 정리는 이제 저희가 할게요!" + 완등 초크볼 + 프로필 링크
