## 슬라이드 구성
1. 표지 — "내 완등 영상, 스토리에만 있음" + 폰뒤짐 초크볼
2. 공감 재현 — 스토리 보관함→없음 / 갤러리→4,832개 / 단톡→유효기간 만료 → "결국 못 찾았어요,,"
3. 페인 확장 — "어제 뭐 풀었는지도 기억이 안 나요..?" (1탄 콜백)
4. 저장 슬라이드 — "기록 안 하면 생기는 일" 3가지 + 📌 저장
5. 해결1 — "완등 기록에 영상까지 같이!" + 문제 상세 실화면(완등 영상·베타·체감·완등률) + 유레카
6. 해결2 — "색깔별로 차곡차곡 쌓여요" + 이번 셋 문제 리스트 실화면 + 등반 초크볼
7. 보너스 — "같은 색끼리도 쉬운 순으로 정렬돼요" + 쉬운 순/어려운 순 토글 + 🍯 꿀문제 크롭
8. CTA — "스토리 말고 여기에 쌓아요!" + 완등 초크볼 + 프로필 링크 (+ "크루 없어도 그냥 쓰면 돼요" — 5탄 떡밥)

## 캡션

```
완등하고 신나서 스토리 올렸는데,
한 달 뒤에 "그 영상 어딨지" 하면서
보관함 무한 스크롤 해보신 적,, 있으시죠?

벽은 매주 바뀌니까 (1탄에서 말씀드렸던 그 얘기요)
내가 뭘 풀었는지도 같이 사라지더라고요.
늘고 있는 건지 아닌지도 모르겠고요,,

그래서 만들어봤어요.
· 완등 기록에 영상을 문제별로 붙여둘 수 있어요
· 색깔별로 기록이 쌓이고, 크루 베타(공략)도 보여요
· 같은 색끼리도 쉬운 순으로 정렬돼요 (꿀문제엔 🍯)

웹에서 무료로 써볼 수 있어요 → 프로필 링크
저장해두시고, 맨날 영상 찍어주는 크루원 태그해주세요 🙌
여러분은 완등 영상 어디에 모으세요? 댓글로 알려주세요!
```
