# 릴스 01 촬영 체크리스트

> 릴스 구조: [릴스01_런칭_컨셉.md](릴스01_런칭_컨셉.md) 참고. 앱 화면 녹화(C)는 Claude 담당.

## 공통 규칙

- **세로(9:16)**, 기본 카메라, 1080p 60fps 이상 (4K 가능하면 4K)
- 각 컷 **최소 5초씩** 촬영 (실제 사용은 1~3초 — 편집 여유 필수)
- 같은 컷 **2~3테이크**
- 얼굴 노출 X: 가슴 아래 높이, 손·발·홀드·벽 위주
- 손떨림 보정 ON, 암장에선 조명 밝은 벽 위주로

---

## A. 집에서 (10분)

### A1. 훅 컷 ★ 제일 중요 — 릴스 0–2초
- 책상 위: **기계식 키보드(또는 맥북) 옆에 초크백**을 나란히
- 노트북 화면엔 코드 에디터 띄워두면 금상첨화
- 위에서 45도 각도, 폰을 천천히 앞으로 밀며(줌인 느낌) 3~5초
- 이 컷 위에 자막 "개발자들은 왜 다 클라이밍장에 있을까"가 올라감

### A2. 인스타 뒤지는 손 ★ 두 번째로 중요 — 릴스 11–17초
- 본인 인스타 프로필 → 스토리 하이라이트/릴스 탭을 **위아래로 빠르게 왔다갔다** 스크롤
- 찾다가 멈칫, 다시 스크롤 — 살짝 답답한 손놀림 연기
- 폰 화면 밝기 최대, 폰 든 손+화면이 같이 보이는 앵글 (어깨 너머 or 책상 위)
- 8~10초 길게 한 번에

## B. 암장에서 (20분)

### B1. 벽 앞 패닝 — 릴스 2–6초
- 벽 하나를 왼쪽→오른쪽 **천천히** 패닝 3~5초 (빨리 돌리면 못 씀)

### B2. 홀드 잡는 손 클로즈업 — 릴스 2–6초 + 마지막 정지컷
- 초크 묻은 손이 홀드를 **잡는 순간** 클로즈업
- 서로 다른 색 홀드로 2~3번 (마지막 CTA 정지컷도 여기서 골라 씀)

### B3. 벽 4개 각각 1~2초 — 릴스 6–11초 "벽은 매주 하나씩 바뀜"
- 각 벽 앞에서 짧게 고정샷 (or 걸으며 훑기)
- 벽마다 문제 색이 다르게 보이는 구도면 최고
- 벽이 4개 안 되는 암장이면 섹션 4곳으로 대체

### B4. (선택) 완등 순간
- 탑 홀드 잡는 손, 또는 완등 후 매트에 내려온 발/털썩 앉는 컷

### B5. (선택) 초크 팡
- 초크백에 손 넣었다 빼며 먼지 확 나는 컷 — 마스코트 '완등 팡' 모션과 톤 맞춤용

## C. 앱 화면 녹화 (Claude 담당) ✅ 완료

`public/brand/screencap/` — 폰 목업 프레임 포함, 780×1780 고해상도.

- `c1-home` (9초): 홈 — 확정 카드 → 가야 할 암장 스크롤 — 릴스 17–24초
- `c2-poll` (12초): 투표 — X-체크(18일) → 겹침 "N명 ✕" → 응답 제출 — 릴스 24–31초
- `c3-map-calendar` (10초): 탐색 지도(수도권 암장 핀) → 캘린더(크루 일정·나도 갈래요) — 여분 컷
- 각 클립 앞 2~3초는 로그인 화면이라 **트림하고 사용**. 폰 프레임 밖 베이지 여백은 크롭.
