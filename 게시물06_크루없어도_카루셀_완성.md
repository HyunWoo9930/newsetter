# 게시물 06 — "저 크루 없는데요,,?" 카루셀 (완성본 · 시리즈 5탄)

> 이미지: `public/brand/carousel5/slide-01.png` ~ `slide-08.png` (1080×1350, 4:5)
> 오디언스 확장 편 — 1~4탄에서 소외됐던 혼클족(크루 없는 클라이머)에게 직접 말 걸기. 개인 모드 소개.
> 재생성: `scripts/reels/carousel5-slides.html` + `render-slides.mjs`
> ⏱ 게시 순서 유동적: 댓글·DM에서 "크루 없어도 되나요"가 2회 이상 나오면 즉시 앞당길 것.

## 슬라이드 구성
1. 표지 — "크루 크루 하는데 말이죠,," + 말풍선 "저 크루 없는데요,,?" + 인사 초크볼
2. 공감 — "클라이밍은 좋은데 크루는 부담스러워요" + 등반 초크볼
3. 페인 — "이게 전부 내 머릿속에 있어야 해요" (언제 갔는지/벽 바뀜/다음 암장)
4. 저장 슬라이드 — "혼자 다니는 분들을 위한 기능 3가지" + 📌 저장
5. 해결1 — "가입하면 바로 나의 클라이밍!" + 개인 모드 홈 실화면 + 유레카
6. 해결2 — "★ 암장 간 지 몇 주째인지 열자마자 보여요" + '가야 할 암장' 크롭(간 지 6주·5주 배지) — 1탄 쿨타임 콜백
7. 해결3 — "지도에서 안 가본 암장 찾기!" + 탐색 지도 실화면(수도권 핀) + 크루 확장 떡밥(초대 코드)
8. CTA — "혼자여도, 크루여도 계속되니까요!" + 완등 초크볼 + "가입 없이 구경만 해도 돼요"

## 캡션

```
1탄부터 보면서 "저 크루 없는데요,," 하신 분들,
사실 제일 많다는 거 알아요 ㅎㅎ

혼자 다니면 편한데, 대신
어느 암장 언제 갔는지, 어디가 벽 바꿨는지, 다음엔 어디 갈지
이게 전부 내 머릿속에만 있어야 하더라고요.

그래서 크루 없이도 되게 만들어놨어요.
· 즐겨찾기 ★ 하면 나만의 홈 암장이 돼요
· 간 지 오래된 곳은 '가야 할 암장'에 자동으로 떠요
· 지도에서 안 가본 암장도 찾을 수 있고요

나중에 크루가 생기면 초대 코드만 뿌리면 되고,
기록은 그대로 이어져요.

웹에서 무료예요 → 프로필 링크
혼자 다니는 클라이밍 친구 태그해주세요 🙌
여러분은 혼자파세요, 크루파세요? 댓글로 알려주세요!
```

## 운영 메모
- CTA가 시리즈 중 제일 가벼움("구경만 해도 돼요") — 팔로우·가입 전환 목적.
- 첫 댓글: "크루 없이 쓰다가 크루 만든 분이 벌써 계세요 ㅎㅎ"
- 시딩: 개발자 커뮤니티보다 클라이밍 커뮤니티의 혼클 글에 맞음.
- 지표: 팔로우 전환 + 링크 클릭. 댓글 질문(혼자파/크루파)으로 댓글 물꼬.
- 다음(6탄 후보): "파랑이 파랑이 아님" — 난이도 지도 기능 출시 후, 환산표 리드마그넷과 동시 공개.
